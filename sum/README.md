## Sum of two numbers
Write a program `to add two numbers`.

### User-experience
```
To add two numbers.
Enter the first number: 2
Enter the second number: 3
The sum of 2 and 3 is 5.
```
[![asciicast](https://asciinema.org/a/UPtfGEn3ciG4ix8GnaC2Jg2S3.svg)](https://asciinema.org/a/UPtfGEn3ciG4ix8GnaC2Jg2S3)

### Correctness
Run you program and verify that your ouput is in line with the user experience.

### Run Tests
Run the test to `verify the correctness` of your program.
```
check50 rohinibarla/c/problems/sum
```

### Coding Style
Check coding style errors
```
style50 sum.c
```

### Submit
Once all test cases pass, `submit your code`.
```
submit50 rohinibarla/c/problems/sum
```
